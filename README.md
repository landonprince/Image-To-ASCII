# Image To ASCII Converter
**By: Landon Prince (5/9/2024)**

Welcome! This program uses the OpenCV library to allow users to convert nearly any image to high-quality ASCII art.
The ASCII art is generated and returned in a text file. Simply enter the file path of the image you want to convert and enjoy the ASCII art!
